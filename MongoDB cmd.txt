Il faut tout d'abord savoir que :

	MongoDB >=< MySQL
	Database >=< Database
	Collection >=< Table
	Document >=< Row
	Field >=< Column

	"mongod" : pour démarrer le serveur MongoDB nous exécutons la commande.
	"mongo" : pour démarrer le client MongoDB nous exécutons la commande.

A partir du client MongoDB nous pouvons éxécuter les commandes suivantes :

	"show dbs" : pour afficher les bases de données.
	"use myDataBase" : pour créer une base de données et l'utiliser.
		admin   0.000GB
		config  0.000GB
		local   0.000GB

	"db.persons.insert({"prenom": "Hamza"})" : pour insérer une donnée dans une table "persons" existante ou nouvelle.
		WriteResult({ "nInserted" : 1 })

	"db.createCollection("persons")" : pour créer une nouvelle collection.

	"show dbs" : pour afficher les bases de données.
		admin       0.000GB
		config      0.000GB
		local       0.000GB
		myDataBase  0.000GB

	"use myDataBase" : pour switcher vers une base de données.
		switched to db myDataBase

	"show collections" : pour afficher les tables := collections.
		persons

	"db.persons.find()" : pour afficher les données.
		{ "_id" : ObjectId("6048d9839b964c6ca2426168"), "prenom" : "Hamza" }

	"db.runCommand({dropDatabase:1})" : pour supprimer la base de données actuelle, la commande générique est "db.runCommand({dropDatabase:1, writeConcern:<document>, comment:<any>})".
		{ "dropped" : "mydb", "ok" : 1 }

	"db.dropDatabase()" : pour supprimer la base de données actuelle.
		{ "dropped" : "mydb", "ok" : 1 }

On peut aussi supprimer une base de données directement sans devoir passer par le client MongoDB en exécutant la commande : [mongo <dbname> --eval "db.dropDatabase()"].

	"mongoimport --db mydb --collection mycoll --file C:\Users\HEALTHINNOVATION\desktop\file.txt" : pour importer un fichier de type json.

	"mongoimport --db mydb --collection mycoll --file C:\Users\HEALTHINNOVATION\desktop\file.csv --type csv --headerline" : pour importer un fichier de type csv.
	ou bien :
	"mongoimport -d mydb -c mycoll --file C:\Users\HEALTHINNOVATION\desktop\file.csv --type csv --headerline" : pour importer un fichier de type csv.
